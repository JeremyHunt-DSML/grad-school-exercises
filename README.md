# Data Science Coursework & Exercises

Welcome to my academic portfolio. This repository contains practical exercises from my Master of Science in Data Science program at Bellevue University. The work here focuses primarily on Python programming, machine learning algorithms, and predictive modeling.

## Table of Contents

* [**IMDb Movie Review Sentiment Analysis**](./imdb-sentiment-analysis/): An end-to-end Natural Language Processing (NLP) text classification project. It demonstrates text preprocessing, TF-IDF vectorization, and the evaluation of multiple machine learning models (Logistic Regression, Naive Bayes, LinearSVC, and Random Forest) to predict review sentiment.
